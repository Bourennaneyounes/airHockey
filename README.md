# Air Hockey

## Description

Air Hockey is a classic tabletop game inspired by Pong, where two opposing players compete to score goals against each other. The game is played on a low-friction table and involves using two hand-held discs and a puck. It's a fast-paced and exciting game that tests your reflexes and strategy.

## Gameplay

- Control your paddle (disc) using the keyboard.
- The objective is to score goals by hitting the puck into your opponent's goal while defending your own.

## Technologies Used

- OpenGL
- C++

### Key Controls

- Player 1:
  - U: Move paddle up
  - H: Move paddle left
  - J: Move paddle down
  - K: Move paddle right

- Player 2:
  - G: Move paddle up
  - V: Move paddle left
  - B: Move paddle down
  - N: Move paddle right
## Installation
